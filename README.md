## üíª How to Run Locally

If you want to run this project on your local machine:

1.  **Prerequisites:** Ensure you have [Ruby](https://www.ruby-lang.org/en/downloads/) and [Jekyll](https://jekyllrb.com/docs/installation/) installed.

2.  **Clone the repository:**
    ```bash
    git clone https://github.com/lchauhan09/lchauhan.git
    cd lchauhan
    ```

3.  **Install dependencies:**
    ```bash
    bundle install
    ```

4.  **Run the server:**
    ```bash
    bundle exec jekyll serve
    ```

5.  **Open in Browser:**
    Navigate to `http://localhost:4000` to view the site.
